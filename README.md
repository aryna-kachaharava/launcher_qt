Разработала экосистему приложений, состоящую 
из Главного лаунчера и четырех независимых модулей (утилит и игр). Все 
компоненты должны иметь графический интерфейс, реализованный на 
библиотеке Qt (Widgets). Сначала вы можете увидеть сам код некоторых приложений (Calculator, Notes, launcher). 
Также представлено итоговое приложение с такими компоненатми как лаунчер, калькулятор, заметки, игра 'Коровы и быки', игра 'Крестики-Нолики'
