#  Экосистема приложений Qt

Разработанная экосистема приложений, состоящая из **Главного лаунчера** и четырех независимых модулей (утилит и игр). Все компоненты имеют графический интерфейс, реализованный на библиотеке Qt (Widgets).

##  Состав экосистемы

| Компонент | Тип | Описание |
|-----------|------|-------------|
| **Launcher** | Ядро | Главный лаунчер для запуска всех приложений |
| **Calculator** | Утилита | Калькулятор для вычислений |
| **Notes** | Утилита | Приложение для заметок |
| **Коровы и быки** | Игра | Логическая игра по отгадыванию чисел |
| **Крестики-нолики** | Игра | Классическая игра 3x3 |


## Технологии

- **Язык:** C++17
- **Фреймворк:** Qt 5/6 (Widgets)
- **Сборка:** CMake / qmake
- **Графика:** Qt Widgets (QWidget, QPushButton, QLineEdit и др.)

### Требования
- Qt 5.12 или новее
- Компилятор с поддержкой C++17
- CMake 3.10+ (опционально)
